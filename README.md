# 🎫 Simulador de Chamados - Help Desk

Sistema de linha de comando desenvolvido em Python para simular a abertura e registro de tickets de suporte técnico. Ideal para demonstrar conhecimentos básicos de lógica de programação e manipulação de arquivos aplicados à rotina de um Service Desk.

## 🚀 Funcionalidades
- Solicita nome do usuário, setor e descrição do problema.
- Registra automaticamente a data e hora do chamado.
- Salva os dados em um arquivo `chamados.txt` para consulta posterior (log de eventos).

## 🛠️ Tecnologias Utilizadas
- Python 3
- Módulos: `datetime`

## ▶️ Como Executar
1. Certifique-se de ter o Python instalado em sua máquina.
2. Clone este repositório ou baixe o arquivo `main.py`.
3. Execute o comando no terminal:
   ```bash
   python main.py
