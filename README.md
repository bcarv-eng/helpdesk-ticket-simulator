# Simulador de Chamados - Help Desk

Sistema desenvolvido em Python para simular o registro de tickets de suporte técnico. O objetivo é demonstrar o entendimento do fluxo básico de um Service Desk, incluindo coleta de informações do usuário, registro de data/hora e persistência dos dados em arquivo de log.

## Funcionalidades

- Entrada de dados: nome do usuário, setor e descrição do problema.
- Geração automática de timestamp no formato brasileiro (dd/mm/aaaa hh:mm).
- Gravação dos chamados em arquivo `chamados.txt` (modo append).
- Exibição do total de chamados registrados após cada inserção.

## Tecnologias utilizadas

- Python 3 (módulo padrão `datetime`)

## Como executar

1. Certifique-se de ter o interpretador Python instalado.
2. Faça o download ou clone este repositório.
3. No terminal, execute: 
python main.py
4. Siga as instruções apresentadas na tela.
