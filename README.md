# Simulador de Chamados - Help Desk

Sistema simples em Python para simular a abertura e registro de chamados de suporte técnico. Os dados são salvos em arquivo `.txt` com data e hora.

## Funcionalidades
- Coleta de nome, setor e descrição do problema.
- Registro com timestamp.
- Persistência em `chamados.txt`.

## Como executar
```bash
python main.py
